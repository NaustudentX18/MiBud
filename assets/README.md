# Assets

Visual assets used in [`README.md`](../README.md). Everything here is
**reproducible** — re-run `python assets/generate.py` from the repo root and
you'll get the same PNGs back.

| `_pytest_output.txt`  | input for tests.png | `pytest -v` capture   |

## Regenerating

```bash
# Capture a fresh test run (optional but recommended before a release)
pytest tests/ -v --no-header 2>&1 | tail -45 > assets/_pytest_output.txt

# Build every PNG
python assets/generate.py
```

The generator pulls fonts from the system:

* `DejaVu Sans` / `DejaVu Sans Mono` — body / monospace text
* `Noto Color Emoji` — the personality emoji glyphs

If those aren't present (`apt install fonts-dejavu fonts-noto-color-emoji`)
the script falls back to a non-color emoji rendering.
